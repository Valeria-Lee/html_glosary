Glosario de html

Etiquetas

<!DOCTYPE HTML>

Declaracion

Indica que documento espera el navegador.

Usada para declarar el doctype.

<!DOCTYPE HTML>

<html>

Etiqueta

Es la ruta de un archivo HTML.

Usada para contener todas las demas etiquetas HTML.

<html></html>

<head>

Etiqueta

Es un contenedor para la metadata.

Usada para definir datos meta tales como titulo, estilos, scripts, etc.

<head></head>

<body>

Etiqueta

Es un contenedor para los contenidos del documento.

Usada para definir el contenido que se renderiza de una pagina web.

<body></body>

<title>

Etiqueta

Es el titulo de la pagina

Usada para definir el titulo en los resultados del navegador, en la toolbar y en favoritos.

<title></title>

<p>

Etiqueta

Es el texto de un parrafo.

Usada para definir un bloque de texto.

<p></p>

<header>

Etiqueta

Es un elemento que contiene un grupo de contenido introductorio.

Usada para barras de busqueda, navegacion, tablas de contenido, logo del sitio, etc.

<header></header>
