<!DOCTYPE HTML>

Declaracion

Indica que documento espera el navegador.

Usada para declarar el doctype.

<!DOCTYPE HTML>

<html>

Etiqueta

Es la ruta de un archivo HTML.

Usada para contener todas las demas etiquetas HTML.

<html></html>

<head>

Etiqueta

Es un contenedor para la metadata.

Usada para definir datos meta tales como titulo, estilos, scripts, etc.

<head></head>

<body>

Etiqueta

Es un contenedor para los contenidos del documento.

Usada para definir el contenido que se renderiza de una pagina web.

<body></body>

<title>

Etiqueta

Es el titulo de la pagina

Usada para definir el titulo en los resultados del navegador, en la toolbar y en favoritos.

<title></title>

<p>

Etiqueta

Es el texto de un parrafo.

Usada para definir un bloque de texto.

<p></p>

<header>

Etiqueta

Es un elemento que contiene un grupo de contenido introductorio.

Usada para barras de busqueda, navegacion, tablas de contenido, logo del sitio, etc.

<header></header>

<nav>

Etiqueta

Es un conjunto de enlaces de navegacion.

Usada para secciones primarias de navegacion.

<nav></nav>

<main>

Etiqueta

Es el contenido principal del documento.

Usada para contenido unico. Solo puede haber uno en el documento.

<main></main>

<aside>

Etiqueta

Es una porcion del documento relacionada al contenido principal.

Usada para informacion adicional como notas, contenido complementario o links relacionados.

<aside></aside>

<footer>

Etiqueta

Es el pie de pagina.

Usada para mapa del sitio web, derechos de autor, informacion de contacto, etc.

<footer></footer>

<h1>, <h2>, <h3>, <h4>, <h5>, <h6>

Etiqueta

Son las etiquetas de encabezado.

Usadas para crear una estructura jerarquica entre titulos y subtitulos.

<h1></h1> ... <hN></hN>

<span>

Etiqueta

Es un contenedor para agrupar contenido en linea.

Usada para facilitar el agregado de estilos o de codigo.

<span></span>

<a>

Etiqueta

Es un hipervinculo.

Usado con el atributo href para hacerlo activo. <a> viene de anchor.

<a href=""></a>

<ol>

Etiqueta

Es una lista ordenada.

Usado para un grupo de items numerados de forma secuencial. Debe de tener al menos un elemento <li>.

<ol></ol>

<ul>

Etiqueta

Es una lista desordenada.

Usado para un grupo de items sin order o secuencia especifica.

<ul></ul>

<dl>

Etiqueta

Es una lista descriptiva.

Usado para contener una lista de terminos emparejados con sus descripciones.

<dl></dl>

<li>

Etiqueta

Es un elemento de una lista.

Usado como un "list item" ya sea de una lista no ordenada (<ul>) o una lista ordenada (<ol>).

<li></li>

<dt>

Etiqueta

Es un termino o nombre en una lista descriptiva.

Usado para especificar un termino dentro de una lista descriptiva.

<dt></dt>

<dd>

Etiqueta

Es una descripcion o definicion de un termino o nombre en una lista descriptiva.

Usado para contener parrafos, imagenes, listas, etc.

<dd></dd>

<table>

Etiqueta

Es una representacion tabular de informacion.

Usado para contener uno o mas header(s), celda(s) y tabla(s).

<table></table>

<tr>

Etiqueta

Es una fila en una tabla de HTML.

Usado para contener uno o mas elementos <th> o <td>.

<tr></tr>

<td>

Etiqueta

Es una celda de informacion estandar.

Usado para contener datos ya sea imagenes, texto u otros elementos.

<td></td>

<th>

Etiqueta

Es el encabezado de una tabla.

Usado para definir una celda como encabezado de un grupo de celdas.

<th></th>

<thead>

Etiqueta

Es un contenedor para todo el contenido de los encabezados.

Usado para proporcionar un significado semantico a la estructura de la tabla. Puede contener elementos "th".

<thead></thead>

<tbody>

Etiqueta

Es un contenedor para el contenido principal de la tabla.

Usado para agrupar informacion de la tabla en una o mas filas.

<tbody></tbody>

<tfoot>

Etiqueta

Es un conjunto de filas que se definen como el pie de la tabla.

Usado para resumir el contenido de las columnas en una tabla.

<tfoot></tfoot>

<img>

Etiqueta

Es una imagen embebida en el documento.

Usada con el atributo src en el que se adjunta el enlace hacia la imagen referenciada.

<img src="">

<audio>

Etiqueta

Es un archivo de audio embebido con renderizado en linea.

Usada para reproducir audio. Tiene atributos como: autoplay, muted, controls para un mejor control. Se usa el atributo src para especificar el archivo.

<audio></audio>

<video>

Etiqueta

Es contenido en formato de video.

Usado con uno o mas elementos <source>.

<video></video>

<source>

Etiqueta

Es un elemento de media.

Usado para especificar algun medio: <video>, <audio>, <imagen>.

<source>

<iframe>

Etiqueta

Es una pagina web o contenido en un sitio embebido.

Usado para definir una region rectangular para contenido como videos, mapas o contenido externo.

<iframe></iframe>

<object>

Etiqueta

Es un contenedor para contenido externo.

Usado para plugins o aplicaciones externas, pero tambien es valido para otros archivos multimedia.

<object></object>

<form>

Etiqueta

Es un formulario para recibir entrada del usuario.

Usado para enviar informacion de un usuario al servidor.

<form></form>

<input>

Etiqueta

Es un elemento que define un campo en el que se pueden introducir datos

Usado para compone a los formularios y cuyo contenido se modifica mediante atributos.

<input>

<select>

Etiqueta

Es una lista o menu desplegable.

Usado para recibir entrada del usuario en formularios.

<select></select>

<option>

Etiqueta

Es un elemento en un menu desplegable.

Usado con el atributo "value" para enviar ese valor al servidor.

<option></option>

<textarea>

Etiqueta

Es un control de edicion de texto.

Usada para recolectar texto como comentarios, reviews o cualquier entrada de texto.

<textarea></textarea>

<label>

Etiqueta

Es una etiqueta de texto.

Usada para ser relacionada con una entrada de usuario.

<label></label>
